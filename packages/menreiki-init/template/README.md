## development

``` bash
npm start
```

## deploy

``` bash
npm run build
```

## debug menreiki

### use vscode

just press `F5` to start debugging

### use chrome inspect

``` bash
  npm run debug
```

then type `chrome://inspect` in chrome address bar, clicking  remote target to debug
